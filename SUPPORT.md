# Support

Use GitHub issues for reproducible CLI bugs, documentation gaps, and public feature requests.

Before opening an issue:

- Search existing issues.
- Include the `@merekit/cli` version, Node version, command, flags, and relevant non-secret output.
- Redact tokens, sessions, workspace data, customer data, private URLs, tenant IDs, and account-specific details.

For security vulnerabilities, do not open a public issue. Follow `SECURITY.md` and use GitHub private vulnerability reporting once this repository is public.

For hosted Mere service access, workspace membership, account recovery, billing, private operator workflows, or customer-specific data questions, use the relevant private Mere/Sawfwair support channel. This public repository cannot grant hosted service access or inspect private workspace state.
